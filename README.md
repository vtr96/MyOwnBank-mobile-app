# MyOwnBank

MyOwnBank é um aplicativo mobile nativo para Android, desenvolvido em Java, com o objetivo de funcionar como um gerenciador financeiro pessoal.

## Tema

Gerenciamento de finanças pessoais, com controle de saldo, entradas e despesas.

## Tipo

Aplicativo mobile nativo para Android.

## Tecnologias

- Android Studio
- Java
- XML para construção das telas
